== Description ==

Plugin pro vytvoření tabulky s daty, které pujde snadno porovnat, filtrovat i prohledávat.


== Installation ==

1. stáhněte ZIP balíček s pluginem
2. rozbalte je do složky s pluginy vaší instalace WordPressu
3. povolte plugin v administraci WordPress
